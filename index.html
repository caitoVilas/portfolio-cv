Diseño de Sistemas a gran escala y Arquitectura de Software

actualmente

Udemy

Este curso enseña a diseñar sistemas altamente escalables, disponibles y fiables para millones de usuarios concurrentes. Cubre: Atributos clave: Rendimiento, escalabilidad, fiabilidad y mantenibilidad. Componentes esenciales: Load balancers, API Gateways, caché, CDN, Message Brokers y multi data center. Almacenamiento de datos: Comparativa SQL/NoSQL, índices, replicación, sharding y teorema CAP. Patrones de arquitectura: Microservicios, CQRS, Event Sourcing y arquitecturas multitier. Ejercicios prácticos: Diseño de sistemas como acortadores de URLs, chats, e-commerce y redes sociales. Objetivo: Preparar profesionales para construir sistemas robustos y enfrentar entrevistas de System Design. Requisitos: Conocimientos básicos de programación y diseño de software.
